## 旧世界(Loongnix)

> 来源: `https://bbs.loongarch.org/d/177-loongnixclash`

下载后:
```bash
unzip -x clash1130-loong64.zip
./clash -d path_to_config.yaml
```


## 新世界

### Archlinux

```bash
pacman -S clash
clash -d path_to_config.yaml
```
